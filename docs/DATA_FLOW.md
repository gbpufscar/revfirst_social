# Data Flow

## Main Flow
1. Ingest open calls and trends from X.
2. Normalize and store as candidates.
3. Classify intent and opportunity score.
4. Generate proposed replies.
5. Run brand and cringe validation.
6. Push valid outputs to approval queue.
7. Publish approved content.
8. Collect metrics and update strategy.

## Telegram Seed Flow
1. Ingest Telegram content.
2. Extract style/topic patterns.
3. Update seed memory.
4. Feed daily post writer.
